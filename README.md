### ğŸ¨ Customization

Add parameters to personalize your widget:

```
?background_color=0d1117&border_color=1DB954&show_status=true
```

**Available Parameters:**
- `background_color` - Background color (hex without #)
- `border_color` - Border color (hex without #)
- `background_type` - `color`, `blur_dark`, or `blur_light`
- `show_status` - Show "Now Playing" text (`true`/`false`)
